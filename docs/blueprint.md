# **App Name**: TrafficFlow

## Core Features:

- Permit Access: View permit documents instantly via a button in the job pack.
- Geo-Fenced Clock-In/Out: Restrict clock-in/out actions to a 100-meter radius of the job site.
- Hazard Logging: Enable quick logging of on-site hazards with a photo and brief description.
- Equipment Checklists: Implement pre- and post-job checklists for TTM equipment verification.
- Digital Completion Sign-Off: Capture a site clearance photo and client/council signature upon job completion.
- Service Reminders: Display service reminders for trucks based on due dates/kilometers using color-coded indicators.
- Defect Reporting: Implement a pre-start checklist to report defects and flag trucks for required checks.
- Fuel Log Integration: Provide a simple form for logging fuel volume and cost to track MPG.
- Utilization Dashboard: Generate key metrics displaying staff and fleet utilization rates.
- Expiry Tracking: Color-coded list of staff certifications expiring soon (e.g., TC, STMS).

## Style Guidelines:

- Primary color: Deep Blue (#22425B) to evoke professionalism and trust.
- Background color: Light Gray (#EAEAEA) to provide a clean and neutral backdrop.
- Accent color: Soft Green (#A2C579) for highlighting important actions.
- Body and headline font: 'Inter', a sans-serif with a machined, objective, neutral look.
- Use clear, consistent icons to represent job status, staff roles, and truck conditions.
- Design a responsive layout that adapts to different screen sizes.
- Use subtle animations for loading states and transitions to improve the user experience.
